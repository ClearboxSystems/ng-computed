This is a basic example of a $computed property. The result here is calculated by the following code: 

            $scope.$computed('computedValue', function() {
                if ($scope.$eval('useCustom')) {
                    return $scope.$eval('customValue');
                } else {
                    return "default value";
                }
            });
When the "use custom" checkbox is checked, the result is the value in the custom value input. When the "use custom" checkbox is left clear, the result is the string "default value".

At the bottom of the page is a log of each watch that is registered/deregistered by the $computed function. As you will be able to see, the watches on "customValue" are registered and deregistered according to the value of the checkbox, while the watches on "useCustom" stay, as the value is always read from the scope.
